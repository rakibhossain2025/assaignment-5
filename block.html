    • Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML দিয়ে html file এর tag এর মধ্যে থাকা কনটেন্ট পাউয়া যায় ।
  • innerText দিয়ে html file এর tag এর মধ্যে থাকা Text পাউয়া যায় । কোন tag পাউয়া যায় নাহ
  • textContent দিয়ে html file এর যেকোনো tag এর ভিতরে থাকা Text Content গুলো পাউয়া যায় । যদি কোন tag থাকে তবে তা পাউয়া যায় না ।

    • Question-3: What is event delegation in the DOM?

    parent element এর মধ্যে Event leastner যুক্ত করা হলে এটি দ্বারা child element গুলোকে নিয়ন্ত্রন করা এবং propgate করা যায়

    Question-4: What is event bubbling in the DOM?

    child element এর মধ্যে Event Listener যুক্ত করা হলে এটি ততক্ষণ পর্যন্ত parent element কে propagate করতে থাকে যতক্ষণ পর্যন্ত না document এ এসে পৌছায়।

    Question-5: How do you create, add, and remove elements using JavaScript?

    document.createElement(), appendChild(), remove() ব্যবহার করে html file এ যে কোন element তৈরি করা , add করা এবং delete করা যায়
